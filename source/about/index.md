## 一、关于我

喜欢写代码，研究各种计算机技术。曾经一度沉迷游戏不可自拔，但也正是因为游戏，才让我接触到了编程，才有了我的今天。从初中起第一次接触 C，便开始了自己的计算机之旅，并被它的魅力所深深吸引。虚拟机，网站，操作系统，C，服务器，程序，每一个名词对当时的我来说都奇妙无比，初中的课余时间挺多，我也可以花更多的时间来折腾计算机。之后陆陆续续学习了许多计算机技术和编程语言，例如HTML， Java，Python，Lisp。高中毕业后，也顺利进入了一所二本院校，并被计算机专业录取，算是达成了自己的一个小心愿。我的人生也进入另一阶段。

在大学里，我被各种计算机技术所吸引，努力地学着各门计算机专业课 ，像编程语言，操作系统，网络原理，数据库，算法与数据结构等。曾经我梦想着考研，读博，但想想读书这么多年，高中，大学，它到底带给我什么，无聊的课程，讨厌的考试，这些都不是我想要的，我应该找回初中那会折腾计算机的热情。总之，为了让我对计算机的热情不至于消失不见，我决定追随我心，去做我喜欢做的事情，我真正热爱的是计算机，是技术。人生很短暂，我需要做我真正热爱的事情。

## 二、关于博客

刚开始用 CSDN 发表博客，但是因为种种原因决定自己搭建一个博客，拥有自己的网站也是我的一个小心愿。刚开始使用优惠价在 阿里云购买了一个服务器，注册了一个域名，搭建了一个 基于 typecho 的动态网站。时间来到了 2021 年，看到续费需要 1000 多，很明显对于现在还没有任何收入的人来说，这是无法负担的，我必须另谋出路。

之后了解了 Github page，觉得这是一个很好的选择，便开始使用 Github page 来作为我的博客服务器，虽然是静态完网站，但对于博客来说也完全足够了。这里给 Github 点个赞，不愧是最受欢迎的开源网站。
本博客域名: https://www.wangjunblogs.com/, 使用了以下技术：
* hexo 作为静态网站生成器
* next 博客主题
* 阿里云注册的域名
* 腾讯云的 CDN 加速服务
* 腾讯云申请的免费 SSL 证书
